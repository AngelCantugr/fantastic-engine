## Daily Routines

### Morning Planning

**Quick (5 min):**
```
Good morning! Help me start the day.

1. Load yesterday's context from memory
2. Show my active projects
3. Suggest 1-3 focus tasks for today

Available time: [X hours]
Energy level: [HIGH/MEDIUM/LOW]
```

**Full (10 min):**
```
Morning planning session.

Please:
1. What was I working on yesterday? (check memory)
2. Show my knowledge graph growth this week
3. List all active projects with their current state
4. Help me plan today based on:
   - Available time: [X hours]
   - Energy level: [HIGH/MED/LOW]
   - Priorities: [What's important today?]
5. Create time blocks for deep work

Then help me set up the first task with sequential thinking.
```

---

### Evening Review

**Quick (5 min):**
```
End of day review.

Today I worked on:
- [Task 1]
- [Task 2]
- [Task 3]

Please save all current context to memory for tomorrow.
```

**Full (10 min):**
```
End of day review for [DATE].

Please help me:
1. What did I accomplish today?
2. What did I learn? (add to knowledge graph)
3. Save context for all active projects:
   - [Project 1]: [current state]
   - [Project 2]: [current state]
4. Note any blockers or questions
5. Suggest priorities for tomorrow

Also update my knowledge graph with today's learnings.
```
